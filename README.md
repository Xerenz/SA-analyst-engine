# SA-analyst-engine
Analyst engine using django and Postgresql

### Basic view

#### index 
page has a search bar where user can query the database for a malware by name
if found then fetch result to malware page
else do a scraping and submit data to db and show result

#### malware 
page displays malware info where each row is a form user can change data
changed data is saved to the db
